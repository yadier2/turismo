
# 💎 App de Turismo
Planea tu próximo viaje a Colombia con nuestra guía oficial de turismo. Descubre destinos turísticos destacados como Bogotá, Medellín, Cartagena entre otros.

Puedes acceder a la aplicacion desde este enlace:
[https://yadier2.github.io/turismo/](https://yadier2.github.io/turismo/).

## Descripcion
Colombia es un país del extremo norte de Sudamérica. Su paisaje cuenta con bosques tropicales, las montañas de los Andes y varias plantaciones de café.
La amabilidad, calidez y pasión de los colombianos por la vida son cualidades que encontrará el turista en todas las latitudes del territorio nacional.
En esta App podrás realizar tus reservas con tranquilidad, para que junto a tú familia disfrutes de los mejores lugares de Colombia.

## Requisitos
Debes de tener instalado NPM y Node.js para su correcta ejecución.

## 🔥 Instalación
1. Instalar las dependencias necesarias a través  del comando (npm install)<br/>
2. Ejecutar el proyecto a través  del comando (npm run start)
